# مستندهای PyPhysicist

به راهنمای جامع خوش‌آمدید! این بخش یک مسیر کامل و قدم‌به‌قدم برای یادگیری، استفاده، و توسعه‌ی کتابخانه‌ی **PyPhysicist** است. این مستندها طوری نوشته شده‌اند که هم برای شروع سریع مناسب باشند و هم برای کاربران حرفه‌ای که دنبال جزئیات دقیق‌تر هستند.

## اینجا چه چیزهایی پیدا می‌کنید؟

- **آشنایی کامل با فلسفه‌ی کتابخانه** و اینکه دقیقاً چه مسائلی را پوشش می‌دهد.
- **نصب دقیق و بدون دردسر** (هم نصب ساده و هم نصب برای توسعه‌دهندگان).
- **شروع سریع** برای اولین محاسبه‌ها در چند دقیقه.
- **مفاهیم کلیدی** مثل نام‌گذاری تابع‌ها، ساختار ماژول‌ها، و کار با واحدها.
- **مثال‌های کاربردی و واقعی** با توضیح مرحله‌به‌مرحله.
- **پرسش‌های متداول** برای رفع ابهام‌های رایج.

## مسیر پیشنهادی برای مطالعه

اگر تازه‌کار هستید، بهتر است ترتیب زیر را دنبال کنید تا همه چیز را منظم یاد بگیرید:

1. [مقدمه](مقدمه.md) — تصویر کلی از کتابخانه، هدف‌ها، و مخاطبان.
2. [نصب و راه‌اندازی](نصب-و-راه‌اندازی.md) — نصب، پیش‌نیازها، و بررسی صحت نصب.
3. [شروع سریع](شروع-سریع.md) — چند خط کد برای اولین محاسبه‌ها.
4. [مفاهیم کلیدی](مفاهیم-کلیدی.md) — اصولی که باعث می‌شوند راحت‌تر کار کنید.
5. [آموزش کامل LaTeX](آموزش-لاتکس.md) — خروجی‌گیری، جایگذاری، محاسبه و توسعه‌ی فرمول‌ها.
6. [مثال‌های کاربردی](مثال‌ها.md) — نمونه‌های واقعی از شاخه‌های مختلف فیزیک.
7. [پرسش‌های متداول](پرسش‌های-متداول.md) — پاسخ به سؤال‌های پرتکرار.

## استانداردهای نوشتاری در این مستندها

برای اینکه محتوای مستندها یکدست و قابل‌اعتماد باشند، این استانداردها رعایت شده‌اند:

- **نام تابع‌ها دقیقاً مطابق API** نوشته شده‌اند (مثلاً `newton_second_law`).
- **پارامترها با نام** نوشته می‌شوند تا خوانایی بالا باشد.
- مثال‌ها تا حد ممکن **واقعی و قابل‌اجرا** هستند.
- نکته‌های کاربردی با قالب **نکته** یا **هشدار** مشخص می‌شوند.

## اگر دنبال چیزی خاص هستید

- برای پیدا کردن یک فرمول، معمولاً کافی است نام فیزیکی آن را بدانید.
- اگر در مورد واحدها مطمئن نیستید، بخش «مفاهیم کلیدی» را ببینید.
- برای کارهای حرفه‌ای‌تر مثل ساخت خروجی LaTeX، به «آموزش کامل LaTeX» سر بزنید.

> اگر اولین بار است که با PyPhysicist کار می‌کنید، پیشنهاد می‌کنم از **مقدمه** شروع کنید و بعد به سراغ نصب و شروع سریع بروید.
