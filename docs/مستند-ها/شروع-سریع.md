# شروع سریع

اگر عجله دارید، این چند خط کافی است تا اولین محاسبه‌ها را ببینید.

## مثال پایه

```python
import PyPhysicist as pp

force = pp.newton_second_law(mass=2.0, acceleration=9.81)
voltage = pp.voltage(current=0.5, resistance=10.0)
energy = pp.relativistic_energy(mass=1.0)

print(force, voltage, energy)
```

**توضیح کوتاه:**
- اول کتابخانه را با نام کوتاه `pp` صدا می‌زنیم.
- هر فرمول یک تابع است و ورودی‌ها همان پارامترهای فیزیکی هستند.
- خروجی هم عدد محاسبه‌شده است.

---

## مثال چندشاخه‌ای

با چند خط می‌توانید از شاخه‌های مختلف استفاده کنید:

```python
import PyPhysicist as pp

# مکانیک
kinetic = pp.kinetic_energy(mass=3.0, velocity=4.0)

# ترمودینامیک
pressure = pp.ideal_gas_pressure(n=1.0, t=300.0, v=0.024)

# موج و اپتیک
wavelength = pp.wavelength(speed=340.0, frequency=170.0)

print(kinetic, pressure, wavelength)
```

---

## استفاده از توابع به‌صورت مستقیم

اگر دوست دارید فقط یک تابع مشخص را ایمپورت کنید:

```python
from PyPhysicist import coulomb_force

force = coulomb_force(q1=1e-6, q2=2e-6, r=0.03)
print(force)
```

---

## خروجی LaTeX برای فرمول‌ها

PyPhysicist می‌تواند معادلات را به LaTeX هم تبدیل کند:

```python
from PyPhysicist import LatexConverter

latex = LatexConverter()
print(latex.symbolic("force"))
print(latex.substitute("force", mass=2.0, acceleration=9.81))
```

**کاربرد:** اگر در گزارش یا مقاله نیاز به فرمول‌های مرتب دارید، این قابلیت کمک زیادی می‌کند.

## نکته‌های سریع

- نام تابع‌ها معمولاً قابل حدس هستند.
- همیشه از آرگومان‌های نام‌دار استفاده کنید تا خوانایی بالا باشد.
- اگر با NumPy کار می‌کنید، بیشتر توابع با آرایه‌ها هم سازگار هستند.

> برای جزئیات بیشتر، به بخش «مفاهیم کلیدی» و «مثال‌ها» سر بزنید.
