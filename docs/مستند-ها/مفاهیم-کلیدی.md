# مفاهیم کلیدی

این چند مفهوم کمک می‌کند موقع استفاده از کتابخانه راحت‌تر باشید و از همه‌ی امکاناتش استفاده کنید.

## 1) نام‌گذاری تابع‌ها

تابع‌ها اغلب با نام فرمول فیزیکی آمده‌اند. مثلاً:

- `newton_second_law` برای قانون دوم نیوتن
- `kinetic_energy` برای انرژی جنبشی
- `ideal_gas_pressure` برای فشار گاز ایده‌آل
- `coulomb_force` برای نیروی کولن

**پس اگر اسم فرمول را بدانید، احتمالاً اسم تابع را هم حدس می‌زنید.**

## 2) ورودی‌ها همیشه واضح هستند

بیشتر تابع‌ها از آرگومان‌های نام‌دار استفاده می‌کنند:

```python
import PyPhysicist as pp

pp.kinetic_energy(mass=2.0, velocity=3.0)
```

یعنی موقع صدا زدن تابع، مشخص می‌کنیم هر عدد مربوط به چه کمیتی است.

## 3) ساختار ماژول‌ها

هر شاخه‌ی فیزیک در پوشه‌ی خودش قرار دارد اما از بیرون لازم نیست درگیر پوشه‌ها باشید.

- `mechanics` → مکانیک
- `electromagnetism` → الکترومغناطیس
- `thermodynamics` → ترمودینامیک
- `waves_optics` → موج و اپتیک
- `relativity` → نسبیت

از سطح بالا همه‌چیز قابل دسترسی است:

```python
import PyPhysicist as pp

pp.momentum(mass=2.0, velocity=5.0)
```

## 4) کار با واحدها (پیشرفته ولی مفید)

کتابخانه یک سیستم سبک برای واحدها دارد. اگر از `Quantity` استفاده کنید، خروجی هم واحددار برمی‌گردد.

```python
from PyPhysicist import Quantity, kinetic_energy

energy = kinetic_energy(
    mass=Quantity(2.0, "kg"),
    velocity=Quantity(3.0, "m/s"),
)
print(energy)
```

> برای پروژه‌های بزرگ‌تر می‌توانید واحدهای خارجی مثل Pint را هم کنار PyPhysicist استفاده کنید.

## 5) ثابت‌های فیزیکی

ثابت‌های مهم از قبل آماده هستند و در سطح بالا در دسترس‌اند:

```python
from PyPhysicist import SPEED_OF_LIGHT, PLANCK_CONSTANT

print(SPEED_OF_LIGHT, PLANCK_CONSTANT)
```

این کار جلوی تایپ اشتباه یا عددهای پراشتباه را می‌گیرد.

## 6) خروجی LaTeX برای گزارش‌ها

اگر نیاز دارید فرمول‌ها را داخل گزارش یا مقاله ببرید، تبدیل به LaTeX ساده است:

```python
from PyPhysicist import LatexConverter

latex = LatexConverter()
print(latex.symbolic("velocity"))
```

## 7) خطاها و مدیریت ورودی‌ها

- اگر پارامتر لازم را ندهید، خطا دریافت می‌کنید.
- برای بعضی توابع مثل مقاومت سری/موازی، ورودی‌ها باید لیست یا چند مقدار باشند.
- اگر از واحدها استفاده می‌کنید، مطمئن شوید واحدها همخوانی دارند.

> نتیجه: با رعایت نام‌گذاری و ورودی‌ها، تجربه‌ی استفاده بسیار روان خواهد بود.
