# مفاهیم کلیدی

این چند مفهوم کمک می‌کند موقع استفاده از کتابخانه راحت‌تر باشید:

## 1) نام‌گذاری تابع‌ها

تابع‌ها اغلب با نام فرمول فیزیکی آمده‌اند. مثلاً:

- `newton_second_law` برای قانون دوم نیوتن
- `kinetic_energy` برای انرژی جنبشی
- `ideal_gas_pressure` برای فشار گاز ایده‌آل

**پس اگر اسم فرمول را بدانید، احتمالاً اسم تابع را هم حدس می‌زنید.**

## 2) ورودی‌ها همیشه واضح هستند

بیشتر تابع‌ها از آرگومان‌های نام‌دار استفاده می‌کنند:

```python
pp.kinetic_energy(mass=2.0, velocity=3.0)
```

یعنی موقع صدا زدن تابع، مشخص می‌کنیم هر عدد مربوط به چه کمیتی است.

## 3) حوزه‌های فیزیک جدا شده‌اند

کدهای مرتبط با هر شاخه‌ی فیزیک در پوشه‌ی خودش است (مثل `mechanics` یا `electromagnetism`).
ولی از بیرون، لازم نیست درگیر پوشه‌ها باشید؛ همه‌چیز از سطح اصلی قابل دسترسی است.

## 4) کار با واحدها (در صورت نیاز)

کتابخانه یک سیستم سبک برای واحدها هم دارد. یعنی اگر با کمیت‌های واحددار کار کنید، خروجی‌ها هم واحددار برمی‌گردند.
این بخش پیشرفته‌تر است و برای شروع لازم نیست نگرانش باشید.

> خلاصه: تا وقتی از اعداد ساده استفاده کنید، مثل یک ماشین‌حساب خیلی تمیز کار می‌کند.
