# نصب و راه‌اندازی

این بخش همه‌ی مراحل نصب را پوشش می‌دهد؛ از نصب ساده برای کاربران عادی تا نصب توسعه‌ای برای کسانی که می‌خواهند کد را تغییر دهند.

## پیش‌نیازها

- **پایتون 3.8 یا بالاتر**
- کتابخانه‌ی **NumPy** (به صورت خودکار نصب می‌شود)

## نصب از PyPI (روش پیشنهادی)

ساده‌ترین روش نصب این است:

```bash
pip install PyPhysicist
```

بعد از نصب، می‌توانید مستقیم در پایتون ایمپورت کنید:

```python
import PyPhysicist as pp
```

### بروزرسانی به آخرین نسخه

اگر قبلاً نصب کرده‌اید و می‌خواهید آپدیت کنید:

```bash
pip install --upgrade PyPhysicist
```

## نصب در محیط مجازی (پیشنهاد برای پروژه‌های جدی)

برای جلوگیری از تداخل نسخه‌ها بهتر است از محیط مجازی استفاده کنید:

```bash
python -m venv .venv
source .venv/bin/activate
pip install PyPhysicist
```

اگر در ویندوز هستید:

```bash
.venv\Scripts\activate
```

## نصب از سورس (برای توسعه‌دهندگان)

اگر مخزن را کلون کرده‌اید و می‌خواهید تغییر بدهید یا توسعه بدهید:

```bash
pip install -e .
```

این روش باعث می‌شود هر تغییری که در کد می‌دهید بلافاصله قابل استفاده باشد.

## بررسی صحت نصب

برای اینکه مطمئن شوید همه‌چیز درست نصب شده است، این قطعه کد را اجرا کنید:

```python
import PyPhysicist as pp

print(pp.newton_second_law(mass=1.0, acceleration=9.81))
```

اگر خروجی حدود `9.81` بود، نصب موفق بوده است.

## رفع خطاهای رایج

- **ImportError:** معمولاً به خاطر نصب نشدن پکیج یا فعال نبودن محیط مجازی است.
- **نسخه‌ی پایتون قدیمی:** PyPhysicist روی نسخه‌های قدیمی‌تر از 3.8 کار نمی‌کند.
- **مشکل در نصب NumPy:** مطمئن شوید pip به‌روز است:

```bash
pip install --upgrade pip
```

> اگر باز هم خطا دارید، معمولاً حذف و نصب دوباره راه‌حل سریع است.
