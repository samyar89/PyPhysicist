# نصب و راه‌اندازی

## نصب از PyPI

ساده‌ترین روش نصب این است:

```bash
pip install PyPhysicist
```

بعد از نصب، می‌توانید مستقیم در پایتون ایمپورت کنید:

```python
import PyPhysicist as pp
```

## کار با سورس (اگر کد را کلون کردید)

اگر مخزن را کلون کرده‌اید و می‌خواهید تغییر بدهید یا توسعه بدهید:

```bash
pip install -e .
```

یا اگر نمی‌خواهید نصب کنید، کافی است ریشه‌ی پروژه در `PYTHONPATH` باشد تا ایمپورت‌ها درست کار کنند.

> پیشنهاد: اگر تازه‌کار هستید، همان نصب معمولی کافی است.
