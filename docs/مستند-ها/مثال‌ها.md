# مثال‌های کاربردی

در اینجا چند مثال واقعی‌تر آورده‌ایم تا ببینید چطور می‌شود از کتابخانه استفاده کرد.

## مثال 1: محاسبه‌ی تکانه

```python
from PyPhysicist import momentum

p = momentum(mass=3.2, velocity=4.5)
print(p)
```

**توضیح:** جرم را در سرعت ضرب می‌کنیم و تکانه می‌گیریم.

---

## مثال 2: فشار گاز ایده‌آل

```python
from PyPhysicist import ideal_gas_pressure

pressure = ideal_gas_pressure(n=1.0, t=300.0, v=0.024)
print(pressure)
```

**توضیح:** تعداد مول، دما و حجم را بدهید تا فشار حساب شود.

---

## مثال 3: ضریب شکست

```python
from PyPhysicist import refractive_index

index = refractive_index(speed_of_light=299_792_458, medium_speed=200_000_000)
print(index)
```

**توضیح:** سرعت نور در خلأ تقسیم بر سرعت نور در محیط.

---

## مثال 4: قانون اهم

```python
from PyPhysicist import voltage

v = voltage(current=0.5, resistance=10.0)
print(v)
```

**توضیح:** ولتاژ = جریان × مقاومت.

---

## مثال 5: مقاومت معادل سری و موازی

```python
from PyPhysicist import resistance_series, resistance_parallel

series = resistance_series(10.0, 20.0, 30.0)
parallel = resistance_parallel(10.0, 20.0, 30.0)

print(series, parallel)
```

**توضیح:** در سری مقاومت‌ها جمع می‌شوند و در موازی از فرمول وارون استفاده می‌شود.

---

## مثال 6: انرژی جنبشی با واحدها

```python
from PyPhysicist import Quantity, kinetic_energy

energy = kinetic_energy(
    mass=Quantity(2.0, "kg"),
    velocity=Quantity(3.0, "m/s"),
)
print(energy)
```

**توضیح:** اگر ورودی‌ها واحددار باشند، خروجی هم واحددار خواهد بود.

---

## مثال 7: خروجی LaTeX برای گزارش

```python
from PyPhysicist import LatexConverter

latex = LatexConverter()
print(latex.symbolic("force"))
print(latex.substitute("force", mass=2.0, acceleration=9.81))
```

**توضیح:** این خروجی را می‌توانید مستقیم در گزارش علمی یا مقاله قرار دهید.
