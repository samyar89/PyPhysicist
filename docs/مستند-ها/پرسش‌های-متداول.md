# پرسش‌های متداول

## آیا این کتابخانه برای استفاده‌ی آموزشی مناسب است؟
بله. هدف اصلی همین است که فرمول‌ها ساده، واضح و سریع قابل استفاده باشند.

## اگر تابع مورد نیازم را پیدا نکردم چه کنم؟
می‌توانید پوشه‌های مربوط به شاخه‌ی فیزیک را بررسی کنید یا مستقیماً در کد جستجو کنید. معمولاً نام تابع نزدیک به نام فرمول است.

## باید واحدها را خودم مدیریت کنم؟
اگر ورودی‌های شما فقط عدد هستند، نیازی نیست واحدها را مدیریت کنید. اما اگر از سیستم واحدها استفاده کنید، خروجی هم واحددار می‌شود.

## چطور می‌توانم کتابخانه را توسعه بدهم؟
بهترین راه این است که یک تابع جدید با نام واضح و آرگومان‌های نام‌دار اضافه کنید و آن را در `__init__.py` اکسپورت کنید.

## آیا پشتیبانی از NumPy دارد؟
بیشتر توابع بر پایه‌ی عملیات ساده هستند و معمولاً با آرایه‌های NumPy هم کار می‌کنند.
