# مقدمه

PyPhysicist یک مجموعه‌ی سبک و کاربردی از فرمول‌های فیزیک در پایتون است. هدف اصلی این کتابخانه این است که **محاسبات فیزیک را سریع، دقیق و خوش‌خوان** کند؛ بدون اینکه هر بار مجبور باشید فرمول‌ها را دستی بنویسید یا دنبال واحدها بگردید.

## چرا PyPhysicist؟

- **سرعت در حل مسائل:** با چند تابع آماده، محاسبات اصلی را انجام می‌دهید.
- **نام‌گذاری واضح:** نام تابع‌ها معمولاً دقیقاً همان نام فرمول است.
- **پوشش چند شاخه‌ی فیزیک:** از مکانیک تا الکترومغناطیس و ترمودینامیک.
- **قابلیت توسعه‌پذیری:** افزودن فرمول‌های جدید ساده است.

## چه چیزهایی داخلش هست؟

در نسخه‌ی فعلی، شاخه‌های زیر پوشش داده می‌شوند:

- **مکانیک:** مثل قانون دوم نیوتن، تکانه، انرژی‌های مختلف، شتاب مرکزگرا.
- **الکترومغناطیس:** قانون اهم، میدان الکتریکی، نیروی کولن.
- **ترمودینامیک:** فشار گاز ایده‌آل، گرمای ویژه، تغییر آنتروپی.
- **موج و اپتیک:** طول موج، بسامد، ضریب شکست.
- **نسبیت:** انرژی نسبیتی، اتساع زمان، انقباض طول، شعاع شوارتزشیلد.
- **ابزارهای تکمیلی:** ثابت‌های فیزیکی، کار با واحدها، تولید خروجی LaTeX.

## این کتابخانه برای چه کسانی مناسب است؟

- **دانشجویان فیزیک و مهندسی** که نیاز به محاسبات سریع دارند.
- **مدرس‌ها و تولیدکنندگان محتوا** که می‌خواهند مثال‌ها را سریع محاسبه کنند.
- **توسعه‌دهندگان پروژه‌های علمی** که به فرمول‌های استاندارد احتیاج دارند.
- **علاقه‌مندان به فیزیک** که می‌خواهند آزمایش‌ها را شبیه‌سازی کنند.

## این کتابخانه چه چیزی نیست؟

برای داشتن انتظار درست، مهم است بدانید PyPhysicist چه کاری انجام نمی‌دهد:

- یک سیستم نمادین (Symbolic) مثل SymPy نیست.
- برای حل معادلات دیفرانسیل طراحی نشده است.
- جایگزین نرم‌افزارهای تخصصی شبیه‌سازی نیست.

> اگر به محاسبات نمادین نیاز دارید، بهتر است PyPhysicist را در کنار SymPy استفاده کنید.

## ساختار کلی کتابخانه

- هر شاخه‌ی فیزیک در پوشه‌ی خودش قرار دارد (مثلاً `mechanics` یا `electromagnetism`).
- توابع اصلی از سطح بالا (`import PyPhysicist as pp`) قابل دسترسی هستند.
- برای فهم کامل API می‌توانید مستندات API انگلیسی را هم بررسی کنید.
